NameDrop

Self-hosted domain drop catching. Monitor expiring domains, track lifecycle states via RDAP, and auto-register when they become available.

GitHub Docker Image

Features

Everything you need to catch expiring domains, no paid APIs required.

🔍

RDAP Monitoring

Query domain status via the free RDAP protocol. No API keys needed for basic monitoring — works with all major TLDs.

🔄

Lifecycle Tracking

Track domains through 8 lifecycle states from registered to available. Full status change history with timestamps.

⏱️

Adaptive Checking

Check frequency increases automatically as domains approach their drop date — from hourly down to every 5 minutes.

⚡

Auto-Registration

Automatically register domains when they become available. Supports Dynadot, Namecheap, GoDaddy, and Gandi via adapter system.

🔔

Multi-Channel Notifications

Get alerted via Webhook, Telegram, Email (SMTP), or ntfy. Includes n8n workflow template for advanced automations.

📦

Bulk Import & Export

Import domains from CSV, text, or JSON files. Export your watchlist anytime. Global search with Cmd+K.

🎨

Dark Dashboard

Clean, data-dense interface with dark and light themes. Status-colored badges, sortable tables, and tag filtering.

🐳

Docker Deploy

Single docker compose up and you're running. SQLite by default — zero configuration needed.

Quick Start

Up and running in under a minute.

docker-compose.yml 
# Clone and start
git clone https://github.com/laurilehto/namedrop.git
cd namedrop
docker compose up -d

# Open the dashboard
open http://localhost:3000
docker-compose.yml 
services:
  namedrop:
    image: ghcr.io/laurilehto/namedrop:latest
    container_name: namedrop
    ports:
      - "3000:3000"
    volumes:
      - namedrop-data:/app/data
    environment:
      - DATABASE_URL=file:/app/data/namedrop.db
      - AUTH_SECRET=${AUTH_SECRET:-changeme}
      - AUTH_PASSWORD=${AUTH_PASSWORD}
    restart: unless-stopped
